# Iris and the Moonlit Garden

A small browser game designed for Iris, age 10.

## Open

Open this file in a browser:

```text
/Users/tian/Documents/Codex/iris-moon-garden/index.html
```

## How to Play

- Move Iris with arrow keys or WASD.
- Collect every star seed.
- In rainbow-door levels, pick up the key first.
- Blue pond tiles slow Iris down.
- Pink hearts restore one life.
- The progress bar and timer track the current run.
- The best full-run time is saved in the browser.
- Avoid the moving shadows.
- After collecting all star seeds, step into the glowing moon gate.

## Files

- `index.html`: game page
- `styles.css`: visual design
- `game.js`: Canvas game logic
